# ClauseAI - Automated Contract Analysis System

**ClauseAI** is an intelligent, multi-agent AI system designed to analyze legal contracts, identify risks, and generate strategic reports for finance, legal, and compliance teams. Powered by Gemini 1.5 Pro and Pinecone Vector DB.

## üöÄ Features
- **Multi-Agent Architecture**: Dedicated agents for Compliance, Finance, Legal, and Operations analysis.
- **RAG Implementation**: Retrieval-Augmented Generation for accurate, evidence-based insights.
- **Strategic Reports**: Generates professional PDF/DOCX reports with risk scoring and citations.
- **Interactive UI**: Executive Dashboard, Streamed responses, and customizable report settings.
